Connect to Network Drives

  1. Double click on the Network Neighbourhood icon, > Select Entire network > Select the machine/group > Right click on the Service / Folder > Select the Map Network Drive > Select Drive Name.
  2. Alternatively, if you know the service, Directly Right click on the My Computer icon > Select Map network Drive > Select Drive Name > Type the service path. (e.g. \\STORAGE\soft)

Connect IGIDR LAN through VPN

  1. Open your browser and type https://150.107.115.242 or https://14.139.112.194
  2. When prompted type your username and password which you used to login on computer centre lab.
  3. Click on CTERM from the screen.
  4. Once you login to CTERM you can use all the software /databases you used to access inside IGIDR.
  5. Alternatively you can download hysecure vpn client or App and install in your computer or tablet. Give any of the above ip as hysecure server address, enter your domain id and password then connect.

Note: Terminal server profiles are temporary. Please do not store your data in terminal server and store them in your allotted storage space after processing. The terminal server profile will be cleared without any notice.

CONNECT TO UNIX SYSTEM USING SSH

  1. Click on Start > All Programs > SSH Secure SHELL, Select SSH Secure Shell Client. or you may click SSH icon on your desktop. Then click on quick connect.
  2. Connect to Unix systems
    • a. In the Host Name field, type system name (example: BRAHMA/REDHAT)
    • b. In the User Name field, type your user name.
    • c. In the Password field, type your password.
  3. You will now be logged in to your home directory.
